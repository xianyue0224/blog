- 架构三部曲：掌握原理 - 独立思考 - 总结反思
- 深度剖析优秀开源项目，由表及里，由浅入深
- 视角切换：多切换到架构师视角，从全局思考问题

## 注意事项
- 整体难度不高，不用担心学不会，坚持打卡写心得
- 优秀的程序员不止能够实现功能，更能读懂别人的代码，读懂别人的想法
